# 📚 Repositori Posttes Praktikum Basis Data A1 2025

Repositori ini berisi hasil praktikum Basis Data A1 tahun 2025 dengan menggunakan tools-tools profesional untuk manajemen database.

## 🛠️ Tools yang Digunakan

| No. | Tool | Deskripsi |
|-----|------|-----------|
| 1 | **DataModeler** | Database modeling dan design |
| 2 | **Laragon** | Local development environment |
| 3 | **TablePlus** | GUI database management |

## 📖 Petunjuk Penggunaan

1. Install semua tools yang diperlukan
2. Clone repository ini
3. Buka project di DataModeler untuk melihat struktur database
4. Jalankan environment dengan Laragon
5. Manage database menggunakan TablePlus
